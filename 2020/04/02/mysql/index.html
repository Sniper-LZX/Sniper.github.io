第一章 json

import json

data_1 = "{'a':1, 'b':2, 'c':3}"
data_2 = '{"a":1, "b":2, "c":3}'
j_data = json.loads(data_2)

输入

type(j_data)
→dict

或者打开一个文件
文件里写有

{
    "a":100,   //必须是双引号
    "b":200    //必须是双引号
}

输入

with open('（文件路径）','r') as f:
    json.load(f)

则会以字典方式输出里面的信息
注意：单双引号在json中还是很重要的

第二章 MySQL数据库基本操作

关系型数据库

（另一种为非关系型数据库，比较流行的有：MongoDB，Redis）

链接登录数据库

mysql -u root -p
u:用户名 p:需要密码

查看数据库 选择数据库 数据库表格 查看表格结构
show databases; use database; show tables; desc tables;
查看表中的数据 <限制数量>
select * from table_name

MySQL数据库查看管理软件：Sequel pro （链接：http://www.sequelpro.com ）
注：只有Mac版本
Windows对应软件：Navicat

  • Host：本地127.0.0.1或远程IP
  • Port：3306（默认端口）

命令行操作数据库

  • 创建数据库
    creat database name;
  • 删除数据库
    drop database name;
  • 加在后面指定字符集
    default charset utf8m64 collate utf8m64_general_ci;

插入数据

insert into 'class'('id','name') 
values('1', '高一三班');

修改数据

update 'class' set 'name' = '高一五班' 
where 'name' = '高一三班';

删除数据

delete from 'class';
where id = '6';

安装MySQLdb库

import MySQLdb

DATABASE = {                        // 多个数据库需要用列表'[ ]'
    'host': '12.0.0.1',             // 如果是远程数据库，此处为远程服务器的ip地址
    'database': 'Examination',
    'user': 'root',
    'password': 'wangwei',
    'charset': 'utf8mb4'            // 防止后面结果results中文处出现问号
}

db = MySQLdb.connect(**DATABASE)    //数据收集
