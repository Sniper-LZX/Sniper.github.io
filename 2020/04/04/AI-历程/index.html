第一部分：SPIDER

爬虫

信息存储在页面上，无法通过接口获得

Python库

  1. requests 用来获取页面内容
  2. BeautifulSoup 分析页面

安装

pip install requests
pip install bs4

一个实例

// 封装成函数，爬取网页，返回链接列表
def get_links(url)
    response = requests.get(url)                    // 如：获得返回<Response [200]>为成功
    response.text                                   // 页面内容
    soup = BeautifulSoup(response.text, 'lxml')     // 解析格式lxml用于可以运行在不同的环境中

    links_div = soup.find_all('div', class_ = "pic-panel")    
    // 寻找指定标签，class是Python中的关键字，所以后面加下划线

    links = [link_div.a.get('href') for link in link_div]    // 每一项都为列表里的一个元素
    /* link_div.a.get('href')            // 获得div中a标签的href属性 */
    return links

url = "http://bj.lianjia.com/zufang/"        // 举例网站：链家网
get_links(url)
第二部分：NumPy

NumPy基本操作

Python科学计算的两个库：

  1. NumPy
  2. Pandas
  3. 绘图工具： matplotlib

扩展：列表可以存储任意类型的数据，数组只能存储一种类型的数据

import array
a = array.array('i', range(10))    // 数据类型为int
// 假如执行
a[1] = 's'        // 试图修改数据为字符型
// 会报错无法执行

数组

现在我们所使用的数组基本上封装在NumPy中

import numpy as np

a1 = np.zeros(10, dtype = int)    //生成数组，默认类型float
a2 = np.zeros((4,4), dtype = int)    //生成4×4数组
np.array()    //与array()相比支持多维数组
np.ones((a,b), dtype) np.full((a,b), data) np.zeros_like(a)，np.ones_like(a)，np.full_like(a, data)
a×b数组, dtype指定类型 a×b数组，data处填写数字 创建与a相同行列的数组

注：未来版本中，1是int64类型的，可以指定一个dtpye，可以用float类型
（如：np.full((2,10),1,dtype=float）

random

1.随机数

np.random.random((3,3))        // 随机数的3×3数组-与random.random的区别
np.random.randn(10)            // 从标准正态分布中返回10个随机数
np.random.rand(10, 4).cumsum(0)            // 随机生成10×4二维随机数，再列求和

# 形成随机数时经常会用到
np.random.randint(0,10,(5,5))    // 0-10随机5×5

2.范围取值

np.arange(0,10,2)    // 第三个是步长
np.linspace(0,3,10)    // 第三个是随机取的个数（这里是10个）
np.eye（10）            // 10×10的单位矩阵

扩展：对应的数据类型有取值范围可以自行搜索查询
如：int32-Integer（-2147483648 to 214748364）

访问数组元素

var[a][b] // 等价var[a, b]
var[:2, :1]  //数组切片
// 注：var[:2][:1]不一样，意为取前两行成为一个数组，再取这个数组的前一行

数组属性

维度 体型 元素个数 类型 每个占字节数 总共字节数
a.ndim a.shape a.size a.dtpye a.itemsize a.nbytes

数组运算

print(a + 10)    // 每个元素加10，-、*同理，多维数组也一样
  1. 操作符和np的函数相对应（自查）
    如：”+” 对应于 np.add
  2. 还可以用数学函数
    如：np.pi == π，np.sin(np.pi)【括号内可以换成带有π的多维数组】

统计类型

  1. sum // 一维数组
  2. np.sum(a) // 多维数组全部数值相加
    np.sum(a, axis=0) // 0：列相加，1：行相加
  3. np.max(a)
    np.max(a, axis=0)

等函数……

notebook使用

  • %timeit；此方法来判断程序的执行效率
    如：%timeit sum(n)，%timeit np.sum(n)（将会看到后者效率更高）

比较

  • a > 3
    会得到多维布尔类型的数组
  • np.all(a>-1)
    a内所有数值全部为True才会返回True

变形

np.reshape(c,d)    // a × b转换c × d数组

排序

np.sort(a)    // 每一行从小到大排，不改变a
a.sort(axis)    // 改变a，axis等于0时行列都排序，axis等于1时只行排序，列顺序不变

拼接

np.concatenate([a,b],axis)
/*
    axis = 0 按行不变竖着拼接
    axis = 1 按列不变横着拼接
*/
第三部分：Pandas

Pandas基本操作

扩展：除了json数据格式之外，还有新的数据格式csv

  • 纯文本，使用某个字符集；
  • 由记录组成（典型的是每行一个记录）；
  • 每条记录被分隔符（Delimiter）分隔为字段；
  • 每条记录都有同样的字段序列。

先在file.csv文件中写入：

id,name,class,score
1,li,高一一班,80
2,song,高一二班,69

csv和DataFrame

import pandas as pd
df = pd.read_csv(csv_file_path)        // 读取csv后缀文件
df = pd.DataFrame()         // 规范表格，有许多参数
df = pd.DataFrame(columns = ['A', 'B', 'C', 'D'])     // 各列列名

此时，df的类型为pandas.core.frame.DataFrame

获取所有列名 索引 取第0项记录 前两行 后两行

索引

df = pd.DataFrame(data, index=['one', 'two'])    
// 按照data建立表格，可指定索引标签（这个标签下无法使用loc[1]之类的数字索引）

忽略索引标签的索引

df.iloc[0]    // 从0开始的实实在在的行数索引
df.ix[4]    // 合并loc和iloc，有数字则数字索引，没有则行数索引

切片式索引

df.loc[:2]
df.ix[:2]
df[:2]    // 可以访问多行数据，但是df[2]是错误的

dataframe中的数组数据

df.values                // 转换为数组数据
df.数学                     // 等价于df['数学']
df.数学.values
df.数学.value_counts()    // 统计多少人得了多少分
df[['数学', '语文']].head()        // 提取多列将会成为一个子表格，可以使用head函数

map函数

def func(score):
    if score >= 80:
        return "优秀"
    else if score >= 70:
        return "良"
    else if score >= 60:
        return "及格"
    else:
        return "不及格"

df['数学分类'] = df.数学.map(func)    //新增一列按照func函数填值
map(func, data)

扩展：lambda函数（匿名函数）

def func(number):
    return number+10
# 等价于
func = lambda number: number+10    

df.applymap(lambda x: str(x) + '-')        // 对所有的列进行操作如尾部加‘-’

// 根据多列来生成新的一列
df['new_score'] = df.apply(lambda x: x.数学 + x.语文, axis = 1)
